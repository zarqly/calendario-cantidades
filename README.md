# Dietario Taxi

Aplicación web progresiva para gestión de ingresos de taxistas con Firebase.

## Desplegar en Vercel

1. Sube este proyecto a GitHub
2. Ve a https://vercel.com
3. Importa el repositorio
4. Vercel detectará automáticamente la configuración
5. Deploy!

## Instalar como PWA en Android

1. Abre la URL de Vercel en Chrome (Android)
2. Chrome te preguntará: "¿Añadir Dietario Taxi a la pantalla de inicio?"
3. Acepta
4. La app se instalará como una app nativa

## Desarrollo local

```bash
npm install
npm run dev
```

## Firebase

La app ya está configurada con Firebase. Los usuarios pueden:
- Registrarse con email/contraseña
- Iniciar sesión con Google
- Todos los datos se sincronizan en la nube
